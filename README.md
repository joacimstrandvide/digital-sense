# Digital Sensibility

A project for improving digital security with a curated library of tools.

### Written in Svelte

---

## Purpose

Many people rely on apps and services that collect unnecessary data, weaken privacy, or reduce control over their digital lives. 

Digital Sensibility provides:

- A structured library of vetted privacy respecting tools
- Clear descriptions 
- Focus on open source and privacy first software  


---

## Features

- Organized list of tools by category   
- Links to official project websites  
- Focus on transparency and user empowerment

Current categories:

- Password & Account Security  
- Productivity & Writing  
- Email & Communication  
- Web Browsers & Online Privacy  
- Search Engines  
- VPN  
- Operating Systems  
- File Sharing  
- Media  
- Third-Party App Stores
