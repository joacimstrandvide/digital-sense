# Digital Sensibility

A project for improving digital privacy with a curated library of tools.

### Written in Svelte

## Features

- Organized list of tools by category
- Links to official project websites

Current categories:

- Password & Account Security
- Productivity & Writing
- Email & Communication
- Web Browsers & Online Privacy
- Search Engines
- VPN
- Operating Systems
- File Sharing
- Media
- Third-Party App Stores
